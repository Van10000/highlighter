Транслятор

Автор: Смирнов Иван

Описание:
Данное приложение предназначается для перевода файла с кодом в формат html с подсветкой синтаксиса.

Возможности:
Поддерживаются языки python, pascal и java, но есть возможность написать собственный файл с описанием синтаксиса и свой стиль отображения.

Имеется возможность передать программе директорию с кодом.
В этом случае будут обработаны все файлы в директории.
Для результата будет создана отдельная директория.
Так как в директории могут быть другие файлы помимо кода - обо всех ошибках будет сообщено в консоль.
Если включен параметр -q, то сообщений об ошибках не будет.

Имеется возможность определить язык кода по расширению файла, либо задать явно.

Подробнее возможности описаны в help-е.

Требования:
1. python версии 3.5.1 или выше
2. Библиотека dominate для создания html файлов

Подробности реализации описаны в файле декомпозиции.
