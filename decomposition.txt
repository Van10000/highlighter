Файлы с описанием формата для языков python, pascal и java лежат в файлах py.hig, pas.hig, java.hig.
Файлы со стилями css хранятся в py.css, pas.css и java.css соответственно.
В общем виде файл для описания формата/стиля ищется по имени <расширение>.<hig/css>

Логика поиска файлов с описанием нужного языка и логика пробега по директориям находится в main.py
Логика парсинга формата находится в файле LanguageFormat.py
Составлением из текста набора токенов занимается метод tokenize в файле Tokenizing.py
За создание финального html файла из набора токенов отвечает класс Rendering.py

Классы TokenInfo и ParsedToken являются контейнерами для описания токена и для найденного в тексте токена соответственно.

Все тесты расположены в файле tests.py
